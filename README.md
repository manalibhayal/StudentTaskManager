# Student Task Manager

This is a simple Java web application developed using JSP, Servlet and Apache Tomcat.

The project allows students to:
- Add tasks
- View task list

## Technologies Used
- Java
- JSP & Servlet
- Apache Tomcat
- HTML

## How to Run the Project
1. Open the project in Apache NetBeans
2. Configure Apache Tomcat server
3. Run the project on the server
4. Open browser and go to:
   http://localhost:8080/StudentTaskManager1/
